En esta carpeta quiero crear temas para personalizarl visual studio code. He dejado solo para analisis las carpetas con temas (ej, Forest Green o Navy Blue) como asi dos archivos con la personalizacion de colores (settings.Forest Green.code-workspace y settings.Navy Blue.code-workspace). Actualmente no toma la configuracion de los temas, si las toma de los workspace.

En la carpeta tambien hay un archivo (temas.xml) con los colores que quiero generar. Ese archivo de colores tiene como campo Tema el nombre del mismo. La idea es que todos los temas deben cambiar los bordes de visual studio code (titulos, barra de herramientas, barra de notificacion, etc) por el color que en el xml esta definido como principal. En los json de los .code-workspace encontraras el ejemplo que funciona. El centro de code, el editor, el chat, la terminal, etc, deben quedar con la configuracion default oscura. Quiero que generes 4 series de archivos, por la combinacion de 1 tema y 1 workspace y una serie de cada uno con el default del nuevo tema de code Dark+ y otro con GitHub Dark. Hay solo 4 personalizaciones de colores que se pueden tomar si existe un valor que es el que corresponde a la sintaxis del lenguaje, a los literales, y en las comparaciones de differencias las lineas quitadas con Diff2 y las agregadas con Diff2, el resto el default del tema base (dark+ o GitHub Dark).
En una carpeta vsDark y en otra GitHubDark cada tema, y en la carpeta .config cada workspace con su nombre (ejemplo forest.dark.code-workspace, forest.github.code-workspace, mountains.dark.code-workspace, mountains.github.code-workspace, academy.dark.code-workspace, academy.github.code-workspace, etc)
